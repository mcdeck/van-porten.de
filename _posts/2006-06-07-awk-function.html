Just for the hell of it. A little awk function to find a columns id in a
csv file. Quite a handy one I might add:

function find_column(name,  column,i) {
    column = 0;
    i = 1
        while( i != NF )
        {
                if( $i == name )
                {
                    column = i;
            }
                i++
        }
    return column;
}
